Brighton Java

Brighton Java is a group bringing together Java users, enthusiasts and learners. Our monthly meetings will include talks, workshops and socials. We’re mostly based at the Skiff in Gloucester St and meet on the first Wednesday of the month.

We’re interested in the whole world of Java – android development, server frameworks, Groovy, interesting tools, applications, Clojure, development methodologies and so on and so on – anything Java.

If you have any ideas for future sessions please get in touch.

Planning meeting/social - discussing what people want from Brighton Java and ideas for future sessions. This event will be run by David Pashley and takes place at the Eagle Pub.

1st August 2012

Debate: Is Server-Side Java Dead?

Java is an established server technology but how does it compare to more recent, sexier frameworks and technologies? Is Java suited for cutting-edge development on the modern web? We've asked some speakers from the local development community to speak, propose and oppose the motion. Does Java hold its own, or is it time to move over to more seductive new technologies?
